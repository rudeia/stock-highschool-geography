# Market Class

고등학생 경제 교육용 실시간 모의 자산 투자 시뮬레이션 웹앱입니다. 교사는 PC 대시보드에서 방을 운영하고, 학생은 스마트폰 화면에서 PIN과 닉네임으로 입장해 1억 원의 가상 자본으로 거래합니다.

## 현재 구현

- 홈 화면: 수업 개요, 교사/학생 화면 진입, 6자리 PIN, 기준금리, 예금금리 표시
- 교사 로그인: 아이디 `geography`, 비밀번호 `72727272`로 교사용 대시보드 접근
- 방 만료: 방 생성 후 24시간이 지나면 만료 상태로 전환
- 학생 접속 QR: 방별 QR과 접속 링크 복사 버튼 제공
- 방 생성: 새 방을 만들 때 자산별 최초가 후보 3개 중 하나를 랜덤 배정
- 교사용 화면: 라운드 준비, 라운드 시작, 장 마감, 다음 라운드 준비, 일시 정지, 게임 종료, 선택형 이슈 등록, 직접 이슈 입력, 전체 자산 시황판, 거시 지표 카드, 라운드 해설
- 학생용 화면: PIN 입장, 자산/현금/예금/평가금 대시보드, 보유 종목 대시보드, 기준금리/예금금리/부동산지수 설명, 기업·자산 분석 카드, 진행 중 공개 이슈, 결과 추측 안내, 투자 상품 리스트, 라운드별 선 차트, 매수/매도 티켓, 예금/해지 티켓, 장 마감 해설 요약
- 최종 보고서: 게임 종료 후 학생별 거래 로그, 라운드별 이벤트 분석, 자기 평가 입력, PDF 저장 제공
- 데이터 계층: 방 생성, 이슈 등록, 거래 로그, 라운드 로그, 학생 스냅샷 생성 규칙을 `src/lib/classroomStore.js`로 분리
- Supabase 기본 연동: 환경변수 설정 시 방, 라운드 상태, 자산 가격, 공개 이슈, 참여 학생 목록을 Supabase에 저장하고 Realtime으로 갱신
- 이벤트 운영: 이벤트 유형별 2~3개 이슈 선택지 제공, 라운드별 최대 5개 이슈 등록, 라운드 진행 중에는 가격 미변동, 장 마감 시 확률 판정 후 가격 후반영
- 가격 변동: 이슈가 반영된 자산은 최소 10% 이상 움직이고, 이슈 영향이 없는 자산도 장 마감마다 ±5% 이내의 시장 변동을 가짐
- 반복 이슈 충격: 같은 이벤트 유형이 2회 이상 실제 적용되면 관련 자산은 최소 50% 수준의 변동성 확대를 가짐
- 장 마감 판정: 이슈가 반영되면 실제 이벤트 발생 70%, 이벤트 미발생이지만 기대감 선반영 30%로 결과 표시
- 실패 기사: 확률 판정으로 이슈가 적용되지 않으면 실패 기사 제목/본문이 해설과 뉴스 피드에 표시
- 교육 해설: 이벤트별 쉬운 설명, 인과 흐름, 토론 질문을 함께 표시
- 재무 학습: 종목별 가상 기업 설명, 간단 재무표, 재무 신호 색상, 위험 태그, 투자 전 체크리스트, 학생 질문 프롬프트 표시
- 장 마감 대시보드: 학생별 금액 없이 등수만 표시
- 가상 종목 10개: 네오모빌리티, 코어실리콘, 에코에너지, 글로벌오일, 유니버스엔터, 미래푸드, 스카이항공, 대한은행, 메디케어, 한빛인프라
- ETF 2개: S&P 500 ETF, KOSPI 200 ETF
- 부동산 지수 1개: 도시부동산지수
- 선물 2개: 글로벌 원유 선물, 글로벌 곡물 선물
- 채권 2개: 미국 10년 국채, 아르헨티나 국채
- 예금: 기준금리에 연동되는 단순 예금금리, 라운드 진행 시 분기 이자 반영

## Supabase 연결 지점

환경변수가 없으면 로컬 상태 기반 프로토타입으로 동작합니다. 환경변수를 추가하면 기본 수업 진행 데이터가 Supabase Realtime으로 연결됩니다.

- `rooms`: PIN, 현재 라운드, 진행 상태, 생성 시각, 24시간 만료 시각
- `players`: 방 ID, 닉네임, 현금, 총자산, 수익률
- `assets`: 방 ID, 자산명, 자산 유형, 최초가 후보, 현재가, 가격 이력
- `portfolios`: 플레이어 ID, 자산 ID, 보유 수량
- `round_events`: 방 ID, 라운드, 교사 입력 이슈, 템플릿 유형, 경제 원리, 영향 자산, 토론 질문, 자산별 영향값, 반영 확률
- `trade_logs`: 플레이어 ID, 자산 ID, 매수/매도/예금/해지, 주문 금액, 체결 수량
- `round_logs`: 방 ID, 플레이어 ID, 라운드별 자산과 이벤트 분석
- `reflections`: 학생의 잘한 점, 부족한 점, 다음 계획

초기 Supabase SQL은 `supabase/migrations/20260615000000_initial_market_class_schema.sql`에 준비되어 있습니다.
Supabase 읽기/쓰기와 실시간 구독은 `src/lib/supabaseRoomStore.js`에 분리되어 있습니다.
